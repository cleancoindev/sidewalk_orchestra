# Sidewalk Orchestra

An experimental musical app using pose estimation of a live sidewalk video stream.

### [Demo Video](https://youtu.be/Sz0IXDZDiWc)

[![demo](assets/demo.gif)](https://youtu.be/Sz0IXDZDiWc)

## Development

To watch for files changes in `/src`
```
npm run watch
```

To build:
```
npm run build
```
